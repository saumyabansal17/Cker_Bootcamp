//q8
git init creates a new empty git repository in your local folder whereas 
git clone copies an existing remote repository (with all files and history) to your local system.(used to copy paste remote repo)

//q9
git branch is used to list, create, or delete branche.(don't switch the current branch)
git branch: gives you current branch
git branch branch_name: creates a new branch
git branch -d branch_name: deletes a branch

git checkout is used to switch between branches or restores files.
git checkout f1 (switch to f1)
git checkout -b f1 (create and switch to f1)
git checkout main --test.txt (restore the old test.txt(discard local changes))
