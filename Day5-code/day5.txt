// Modes
Normal Mode – navigation & commands (default). Press Esc to return.
Insert Mode – for typing text. Press i.
Visual Mode – for selecting text. Press v (chars) or V (lines).

// Cursor Movement
Arrows or h(left) j(down) k(up) l(right)
Word: w (next word), e (end), b (previous)
Line: 0 (start), ^ (first non-blank), $ (end)

// Editing Commands
Insert: i (before), a (after)
Delete: x (char), dw (word), dd (line)
Undo: u, Redo: Ctrl + r
Copy/Paste: yy (copy line), p (paste after), P (paste before)

// Save & Quit
:w → save
:q → quit
:q! → quit without saving
:wq or ZZ → save and quit
